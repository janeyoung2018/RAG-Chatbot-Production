# Frontend Chatbot Application

This folder contains the frontend application for the RAG chatbot project, built using React.

## Getting Started

To get started with the frontend application, follow these steps:

1. **Install Dependencies**: Navigate to the `frontend` directory and run the following command to install the necessary dependencies:

   ```
   npm install
   ```

2. **Run the Application**: After installing the dependencies, you can start the development server with:

   ```
   npm start
   ```

   This will launch the application in your default web browser.

## Folder Structure

- `src/`: Contains the source code for the React application.
  - `App.jsx`: The main component that manages the chatbot UI.
  - `index.jsx`: The entry point for the React application.

## Additional Information

For more details on how to customize and extend the application, refer to the documentation of React and any libraries used in this project.
